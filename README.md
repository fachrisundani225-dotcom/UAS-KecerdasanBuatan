# UAS Kecerdasan Buatan - Prediksi Customer Churn E-Commerce

Repositori ini dibuat untuk memenuhi tugas Ujian Akhir Semester (UAS) mata kuliah Kecerdasan Buatan. Proyek ini berfokus pada implementasi *machine learning* secara end-to-end untuk memprediksi potensi *churn* (berhenti berlangganan) pelanggan menggunakan algoritma klasifikasi.

---

## 👥 Anggota Kelompok
* **Nama Anggota 1:** [Fachri Sundani] - [2406048]


---

## 📌 Deskripsi Proyek
Proyek ini membangun model klasifikasi AI prediktif yang dapat memberikan sinyal peringatan dini (*alert*) bagi tim CRM (Customer Relationship Management) terhadap konsumen yang menunjukkan indikasi akan meninggalkan platform e-commerce.

* **Dataset:** E-Commerce Customer Churn Dataset (by Ankit Verma via Kaggle).
* **Algoritma Utama:** Decision Tree Classifier & Random Forest Classifier.
* **Metrik Evaluasi Utama:** Accuracy, Precision, Recall, dan F1-Score.
